# マージ済みローカルブランチの削除

リモート（origin のデフォルトブランチ）にマージ済みのローカルブランチだけを削除する。

## 手順

1. **スクリプトの実行**
   リポジトリルートで以下を実行する。

   ```bash
   ./scripts/delete-merged-branches.sh
   ```

   スクリプトは以下を行う：
   - `git fetch origin --prune` でリモートを最新化
   - リモートのデフォルトブランチ（`origin/HEAD`）にマージ済みのローカルブランチを列挙
   - 現在のブランチとデフォルトブランチ名（例: main）は除外
   - 上記に該当するブランチを `git branch -D` で削除

2. **結果の報告**
   - 削除したブランチ名と数をユーザーに伝える。
   - スキップしたブランチ（現在ブランチ・デフォルト・マージ未完了）があれば理由とともにつける。
