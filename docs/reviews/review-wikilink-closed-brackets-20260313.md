## サマリー

Issue #347 の修正。`[[` 入力時の WikiLink サジェストが、`]]` で完全に囲まれたリンク内（例: `[[Page]]` の "Page" 部分）にカーソルがある場合にも発動する問題を修正した。カーソル以降のテキストで `]]` が `[[` より前に出現する場合は「閉じたリンク内」と判定し、サジェストを発動させない。E2E で回帰防止テストを追加し、`WikiLinkSuggestion` に `data-testid` を付与した。
